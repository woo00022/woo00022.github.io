Links
Also known as "Anchors", can link to other websites, other pages, etc. using href attribute to provide the location.
Blockquote
Large, stand alone quote. Indented in. CAN have a citation
Line Break
Needs to be added to add a link break. doesn't need to be closed because you don't add anything Inbetween it. Don't add double line breaks to fake a paragraph, just start a new paragraph tag.
Image
Adds an image to the page! Needs to be sourced with correct location and path, important to have alternate text for accessibility.
Tables
Adds a table container! Useful for displaying data like schedule, spreadsheet, etc. Nests other tags such as table heading, table row, table data cell to add information.
